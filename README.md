# Ffi::Gr

[GR](https://github.com/sciapp/gr) for Ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ffi-gr'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ffi-gr
