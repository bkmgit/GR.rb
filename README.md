# GR

[GR framework](https://github.com/sciapp/gr) for Ruby

## Installation
Install [GR](https://github.com/sciapp/gr/releases).
Set environment variable GRDIR.

```sh
export GRDIR="/your/path/to/gr"
```

Add this line to your application's Gemfile:

```sh
gem 'ffi-gr'
```
